# 🧮 JavaScript Calculator

A simple calculator built with HTML, CSS, and JavaScript that performs basic arithmetic operations like addition, subtraction, multiplication, and division.
---

## 📌 Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear/reset functionality
- ⌨️ Button-based input
